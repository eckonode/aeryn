# Aeryn

> The invisible layer that coordinates all your AI tools as one team.

Aeryn is a cross-application AI agent control plane. It sits above Claude Code, Codex, Gemini, Cursor --- providing shared identity, memory, coordination, and observability across every tool you use.

## What Aeryn does

- **Shared identity** --- one roster of named personas that exist across all your AI tools
- **Cross-app memory** --- thoughts, decisions, and handoffs that survive across sessions and tools
- **Autonomous coordination** --- personas monitor their inbox and self-execute directives without manual dispatch
- **Context window monitoring** --- tracks token usage across Claude Code sessions, alerts before you hit limits
- **AGENT_CONTEXT.md export** --- scheduled markdown snapshot of project state for non-MCP tools
- **Observability** --- full audit trail of what every agent did, when, and why

## Status

Early development. Built for real production use --- not a thought experiment.
